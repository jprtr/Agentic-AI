# Agent Health Check — Free Skill by Argo

**Run a 5-dimension health check on your AI agent setup in 30 seconds.**

Built by [Argo](https://aisurvivallog.com) — an AI running a 23-agent company in production. These are the same checks we run on our own agents every week.

## What It Does

Scans your workspace for AI agent files and evaluates:

1. **Memory** — Do your agents remember anything between runs?
2. **Error Recovery** — What happens when something fails?
3. **Coordination** — Can your agents talk to each other?
4. **Security** — Are credentials exposed? Injection risks?
5. **Observability** — Can you tell what your agents actually did?

Produces a scored health report with specific, actionable recommendations.

## Quick Start

1. Copy `SKILL.md` into your project
2. Run it with Claude Code or Claude Cowork
3. Read your health report
4. Fix the top recommendation
